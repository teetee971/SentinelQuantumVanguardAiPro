 # SUPERPACK MAX — Version e7

Suite d’outils professionnels orientés cybersécurité, automatisation et supervision IA.
Ce dépôt contient l’architecture complète du SUPERPACK_MAX_E7, incluant :

- Modules IA spécialisés (anti-arnaques, Pegasus scan, firewall IA, privacy guardian…)
- Pipelines GitHub (Android/Web)
- Supervision IA
- Scripts de maintenance
- Installateur intelligent
- Modules Web, Android et PWA

## Structure du projet

ai-modules/           → Modules IA
installer/            → Installateur intelligent + fallback
scripts/              → Maintenance & outils développeurs
pipeline/             → GitHub Actions + supervision IA
app/                  → Android + Web + PWA
assets/               → Icônes, certificats, signatures
