L'application est distribuée directement via GitHub pour garantir une souveraineté totale, sans intermédiaire ni collecte de données par un store tiers.

Pourquoi SHA-256 ?

Le fichier SHA-256 permet de vérifier que l'APK n'a pas été altéré pendant le téléchargement. C'est une empreinte cryptographique unique du fichier original.

Vérification de l'intégrité

Téléchargez l'APK et le fichier .sha256, puis exécutez : sha256sum -c fichier.apk.sha256

sha256sum -c SentinelQuantumVanguardAIPro-v1.0.0.apk.sha256

Utilisez la commande ci-dessus pour vérifier l'intégrité du fichier. Vous pouvez également utiliser apksigner pour vérifier la signature.
