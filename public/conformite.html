Gouvernance du Projet

Sentinel Quantum Vanguard AI Pro est un projet open source maintenu activement avec une gouvernance transparente.

  • Code source ouvert et auditable sur GitHub
  • Historique de commits traçable et vérifiable
  • Processus de release documenté
  • Politique de sécurité publiée (SECURITY.md)
  • Signalement responsable des vulnérabilités

Chaîne de Confiance

Chaque release suit un processus de build et de signature vérifié.

1

Code Source

Code versionné sur GitHub, commits signés, historique complet

2

Build CI/CD

Compilation automatisée via GitHub Actions, logs publics

3

Signature APK

Signature cryptographique RSA 4096 bits, keystore sécurisé

4

Checksum SHA-256

Somme de contrôle publiée pour vérification indépendante

5

Distribution

Publication sur GitHub Releases, téléchargement direct

Souveraineté Numérique

Sentinel garantit une indépendance technologique complète.

  • Aucune dépendance à un store tiers (hors Google Play)
  • Données 100% locales sur l'appareil
  • Aucune collecte de données personnelles
  • Aucun analytics ou tracking tiers
  • Build reproductible et auditable
  • Hébergement sur infrastructure souveraine (GitHub)

Distribution Directe

L'APK est distribué directement via GitHub Releases, permettant une vérification indépendante de l'intégrité et de la signature sans intermédiaire.

Conformité RGPD

Sentinel respecte les principes du Règlement Général sur la Protection des Données.

  • Aucune collecte de données personnelles
  • Aucune transmission de données à des tiers
  • Traitement local uniquement
  • Pas de cookies ni de trackers
  • Architecture privacy-by-design

Principe de Minimisation

Sentinel ne collecte aucune donnée car aucune n'est nécessaire à son fonctionnement. Les analyses et protections sont effectuées localement sur l'appareil.
