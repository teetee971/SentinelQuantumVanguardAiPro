Installation

Sentinel Quantum Vanguard AI Pro - Android 12+

1
Télécharger l'APK

Téléchargez l'APK officiel depuis GitHub Releases.

Télécharger APK
2
Autoriser l'installation

Autorisez l'installation depuis des sources externes dans les paramètres Android.

Paramètres > Sécurité > Sources inconnues (ou Apps de sources inconnues)
3
Vérifier l'intégrité (optionnel)

Vérifiez le checksum SHA-256 pour garantir l'intégrité du fichier.

sha256sum SentinelQuantumVanguardAIPro-vX.Y.Z.apk
Remplacez X.Y.Z par la version téléchargée. Comparez le résultat avec le fichier .sha256 disponible sur GitHub Releases.
Pourquoi hors Google Play ?

La distribution directe offre plusieurs avantages pour un logiciel de sécurité :

  • Indépendance vis-à-vis des stores tiers
  • Vérification cryptographique possible (SHA-256, signature APK)
  • Contrôle total de la chaîne de distribution
  • Mises à jour immédiates sans délai de validation
  • Transparence complète sur le processus de build

Version officielle - GitHub Releases

Produit distribué hors Google Play. Installation manuelle requise.
