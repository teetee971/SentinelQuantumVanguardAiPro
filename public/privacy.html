Step 2: Check Network Requests

  1. Open browser DevTools (F12)
  2. Go to Network tab
  3. Reload the page
  4. Verify: Only requests are for static assets (HTML, CSS, JS, images)
  5. Verify: NO analytics, NO tracking, NO third-party domains

Step 3: Check Local Storage

  1. Open browser DevTools (F12)

  3. Verify: ZERO data stored

Step 4: Inspect Source Code

  2. Search for: analytics, tracking, Google Tag Manager patterns, pixel
