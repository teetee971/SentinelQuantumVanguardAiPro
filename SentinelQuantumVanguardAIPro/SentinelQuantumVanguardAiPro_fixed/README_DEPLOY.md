# Deployment Instructions

This repository contains a placeholder structure for Sentinel Quantum Vanguard AI Pro.

1. Install dependencies for both frontend and backend.
2. Run `npm run build` in `frontend/` to build the PWA.
3. Start the backend server with Node.js.
4. Adjust CI/CD scripts as needed to generate APK/EXE via your build system.
5. There is no `.gitmodules` file and no submodule references in this repository.
