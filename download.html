PC (Windows/Linux)

Agent Sentinel pour PC

Protection locale pour Windows et Linux avec surveillance processus, analyse comportementale et blocage des menaces.

Version : 0.9.0-beta

Plateformes : Windows 10/11, Ubuntu 20.04+, Debian 11+

Taille : ~50 MB

Licence : Open Source (Apache 2.0)

Notice : environnement de test

• Installer uniquement sur machine virtuelle ou environnement de test
• Peut générer des faux positifs
• Consommation CPU/RAM variable selon la charge
• Logs détaillés dans /var/log/sentinel/ (Linux) ou C:\ProgramData\Sentinel\ (Windows)

L'agent PC est en développement actif. Publication prévue courant 2025.

Android

Agent Sentinel pour Android

Protection mobile sans root avec analyse APK, détection phishing et filtrage DNS.

Version : 1.0.0-debug

Android : 6.0 (API 23) minimum

Taille : ~15 MB

Type : APK debug (non publié sur Play Store)

Permissions listées

• INTERNET — Communication avec serveurs de mise à jour IOC
• ACCESS_NETWORK_STATE — Vérification connectivité
• QUERY_ALL_PACKAGES — Analyse des applications installées (Android 11+)
• VPN (optionnel) — Filtrage DNS local sans root

APK debug

• Signé avec certificat de développement
• Non optimisé (ProGuard désactivé)
• Logs verbose activés
• Installation manuelle requise (sources inconnues)

Voir le changelog

L'APK de production sera disponible après validation complète des tests de sécurité.
